**Готовый установщик для фильтрации входящих подключений к вашей VPN-ноде, пропускающий только трафик с мобильных операторов (LTE/5G).**

Всё, что вам нужно — запустить одну команду. Красивый интерактивный мастер задаст несколько вопросов и полностью настроит Nginx или Caddy, блокируя всех, кто пытается зайти с домашнего Wi-Fi или проводного интернета. Идеально для Xray, V2Ray или любого прокси-сервера, работающего за Яндекс CDN.

---

## ⚡ Быстрая установка

Установщик интерактивный (задаёт вопросы), поэтому **сначала скачайте файл, потом запускайте** — так `read` сможет получать ваш ввод с клавиатуры:

```bash
curl -sSL https://raw.githubusercontent.com/pwdrs/Yandex-CDN-only-on-Mobile-Whitelist-Bypass/main/install.sh -o install.sh
sudo bash install.sh
```

> ⚠️ Не запускайте одной командой через `curl | sudo bash` — при таком запуске `stdin` занят потоком скрипта, а не клавиатурой, и интерактивные вопросы (`read -p`) не получат ввод. Скрипт упадёт сразу после первого запроса.

---

## 🔬 Как это работает?

1. **Яндекс CDN** принимает клиентский трафик и пересылает его на ваш origin-сервер, добавляя заголовок `X-Real-IP` с настоящим IP-адресом посетителя.
2. **Nginx / Caddy** на вашем сервере читает этот заголовок и сверяет IP с заранее загруженной базой **мобильных ASN** (Autonomous System Numbers) российских сотовых операторов.
3. Если IP принадлежит, например, МТС, Билайн, МегаФон, Tele2 или другим операторам связи — соединение разрешается.
4. Во всех остальных случаях (домашний проводной интернет, облачные серверы, VPN без CDN) — возвращается **403 Forbidden**.

IPv6-адреса **игнорируются** и пропускаются без проверки. При необходимости отключите IPv6 на ноде.

> ⚠️ **Важно:** фильтрация работает **только** при использовании CDN, который добавляет `X-Real-IP`. Прямые подключения к ноде (минуя CDN) блокируются на уровне конфигурации, либо вы можете закрыть порт файрволом.

## 📦 Требования

- **ОС:** Ubuntu 20.04+ или Debian 11+ (чистая установка или уже с веб-сервером)
- **Веб-сервер:** Nginx **или** Caddy (должен быть предварительно установлен)
- **Python 3** и **curl** (установятся автоматически при необходимости)
- **Активный Яндекс CDN** с поддержкой `X-Real-IP`
- Права `root` для запуска установщика
